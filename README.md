### Introduction

Dự án sử dụng framework laravel 11 để xây dựng nên trang quản lý của
trang thương mại điện tử bán quần áo. Trang quản trị sẽ có các chức năng
như: Đăng nhập, chức năng crud cho sản phẩm, danh mục và brand.
